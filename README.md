# MarioKartTopGenerator

```
help                                    Affiche ce message
quit                                    Quitte le programme

title <str>                             Change le titre
subtitle <str>                          Change le sous-titre
date <str>                              Change la date (format XX/XX/XXXX)
count <value>                           Change le nombre de participants
url <str>                               Change le lien d'invitation discord

load <path>                             Charge le fichier excel du bracket
player <rank> <character> <name>        Change les informations d'un joueur
random                                  Change le fond du rendu
print                                   Affiche toutes les informations concernant le rendu

dir <path>                              Change le dossier où les rendus seront créés
create                                  Créer le rendu
```
